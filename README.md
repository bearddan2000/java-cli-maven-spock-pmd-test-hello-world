# java-cli-maven-spock-pmd-test-hello-world

## Description
A POC for maven app using spock
framework and pmd plugin.

## Tech stack
- java
- groovy
- maven
  - spock
  - pmd

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- pmd report found at bin/target/site

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
